🔧 주요 기술적 특징
================================================================================

아키텍처:
- index.html: 메인 진입점, 다크 테마 사이드바 메뉴, 액티브 메뉴 상태 관리
- iframe 기반 콘텐츠 로딩으로 모듈화
- JavaScript 메뉴 제어 (toggleSubmenu, loadPage, setActiveMenu 함수)
- pages/ 디렉토리: 서비스별 디렉토리 분리
- URL 파라미터 기반 페이지 간 데이터 전달

상세 페이지 시스템:
- 테이블 → 상세 페이지 네비게이션 구조
- work-detail.html: 개별 작업 상세정보 표시
- URL 파라미터로 동적 데이터 로딩
- 뒤로가기 기능 및 액션 버튼 시스템

웍스파이 연동:
- 실시간 지원자 현황 조회
- 프로젝트 모집 등록 팝업
- 작업자 배정 시스템 준비

디자인 컨셉:
- 전문적인 관리자 페이지 디자인
- 다크 사이드바 + 밝은 콘텐츠 영역
- 일관된 색상 체계 (#3b82f6)
- 부드러운 애니메이션 효과
- 심플하고 깔끔한 정보 표시
